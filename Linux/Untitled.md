#Linux命令

```python
pwd: 查看当前所在目录

/ ：根目录

cd xxx: (change directory改变目录)

cd..: 回到上一层目录

clear: 清空当前终端显示的东西

ls: 查看当前目录下的文件

ls -l: 查看当前目录下文件或文件夹的详细信息

ls -a: 查看当前目录下的隐藏文件

touch xxx:创建文件

vim .xxx: 创建隐藏文件并进入vim模式编辑文件

open xxx:打开某个文件

cat xxx :查看某个文件的内容

more xxx:显示一个文件可以逐行或者翻页查看

mkdir xxx(change directory改变目录): 创建一个文件夹

mkdir -p zhongguo/henan/xinxiang: 递归的创建目录，一层一层的全部创建出来。

mv xxx(文件) xxx(文件夹):将文件移至某个文件夹

mv xxx(文件1) xxx(文件2):将文件1覆盖文件2

tree xxx: 以数列的方式展示出文件夹结构

```
